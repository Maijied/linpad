Welcome to Linpad

A powerful, open-source text editor designed for developers and writers.

Download Linpad

Features

  • Syntax Highlighting - Python syntax support for easier readability.
  • Word Count - Displays word and character count.
  • Custom Fonts - Choose your preferred font and size.
  • Dark Mode - Toggle between light and dark themes.
  • Find and Replace - Quickly find and replace text.

Installation

Download the latest release or install it from the Snap Store. Visit the GitHub repository for more information.
